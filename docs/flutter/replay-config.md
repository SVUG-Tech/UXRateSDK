# Session Replay — Flutter

> For concepts, quality presets, and capture modes, see [Session Replay](Session-Replay).

Replay is handled entirely by the native iOS and Android UXRate SDKs. There is no Dart-level API for replay.

## Configuration

All replay settings are managed from the **UXRate dashboard** — no code changes needed:

- Enable/disable replay per project
- Sampling rate
- Masking rules for sensitive fields
- Max session length
