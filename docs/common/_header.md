# UXRateSDK

AI-powered in-app survey SDK for **iOS** and **Android**. Embed conversational surveys into your app with a single line of code.

## Platform Support

**Native SDKs**

| Platform | Language | Min Version | Notes |
|----------|----------|-------------|-------|
| iOS | Swift 5.9+ | iOS 17.0+ | SwiftUI & UIKit supported |
| Android | Kotlin 2.1+ | API 24 (Android 7.0+) | Jetpack Compose & Activities supported |

**Cross-platform SDKs**

| Platform | Language | iOS | Android |
|----------|----------|-----|---------|
| Flutter | Dart 3.0+ | iOS 17.0+ | API 24 (Android 7.0+) |
| React Native | TypeScript | iOS 17.0+ | API 24 (Android 7.0+) |
